# Glossary

## Async Learning

Learning work that can be completed without requiring all participants
to be live at the same time.

## Content Item

A single reusable educational asset, such as a document, slide deck,
video, guide page, or quiz.

## Learning Module

A grouped set of content items intended to be consumed together as one
unit of learning.

## Assignment

A delivery of a content item or module to a learner, player, subgroup,
or team.

## Learner Scope

The audience targeted by an assignment, such as one player, a role
group, or a full team.

## Curated Content

External or pre-existing material selected for use inside `RiftSense`.

## Native Content

Content authored or maintained specifically for `RiftSense`.

## Shared Context

User, team, player, or workspace information that should eventually be
owned by shared ecosystem services rather than permanently duplicated in
`RiftSense`.
